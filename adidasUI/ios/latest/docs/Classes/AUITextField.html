AUITextField

open class AUITextField: UITextField

Adidas UITextField which is IBDesignable allowing it to dynamically show style changes (e.g. textColorName).

By default, this subclass also removes the default border of UITextField.

Hit Area
  • minimumHitArea

    Minimum size to use for the hit area of the text field.

    The default value is CGSize(width: 44, height: 44), which is the recommended minimum size by Apple.

    Note that the superview of the text field should at least have the size of the hit area, otherwise the point(inside:with:) method will never be called anyway.
