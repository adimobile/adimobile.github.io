AUIMessagePopoverController

open class AUIMessagePopoverController: UIViewController

A AUIMessagePopoverController object displays an popover message to the user. After configuring the popover controller present it using the presentViewController:animated:completion: method.

Overview

Listing 1 shows how to configure an popover with a mesage

Listing 1Configuring and presenting an popover “` let popup = AUIMessagePopoverController(message: SIGN UP SUCCESSFUL)

self.presentViewController(popup, animated: true, completion: nil)
