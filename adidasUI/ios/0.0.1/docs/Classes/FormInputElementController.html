FormInputElementController

open class FormInputElementController<Data, Value, Field: UIView & FormInputField>: FormElementController<Data>, UITextFieldDelegate, UITextViewDelegate where Field.Value == Value

The FormElementController for FormInputElement.

This class creates an internal view for hosting a FormInputField. Generally this controller will be the delegate of the FormInputField and fully manages the field.

This class automatically becomes the delegate for UITextField or UITextView fields and handles the delegate methods accordingly.
