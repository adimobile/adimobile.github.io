    Triggers an animation to reveal a container view, together with a piece of text inside the container, using a mask that slowly becomes smaller. This results in the views appearing, while they stay in place.

    Declaration

    Swift

    public class func animateMaskSlide(forContainer containerView: UIView,
                                   textView: UIView?,
                                   delay: TimeInterval = 0,
                                   duration: TimeInterval? = nil,
                                   direction: MaskSlideAnimationDirection = .leftToRight,
                                   inverted: Bool = false,
                                   completion: (() -> Void)? = nil)

    Parameters

    containerView

    The container view, which will be revealed first.
    textView

    The text view inside the container view, which will be revealed once the container is visible.
    delay

    The delay after which to start the animation. The default value is 0.
    duration

    The duration of the animation. By default this will be based on the size of the views.
    direction

    The direction of the reveal animation.
    inverted

    Indicates whether the animation should be inverted. When inverted, the animation will hide the views instead of revealing them. The default value is false.
    completion

    An optional completion block which is called after the animation has finished.
