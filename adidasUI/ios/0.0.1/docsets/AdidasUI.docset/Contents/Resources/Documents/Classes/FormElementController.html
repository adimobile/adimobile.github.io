FormElementController

open class FormElementController<Data>: NSObject

Controller for managing the view of an element in a Form.

Each FormElement has a function for creating an instance of this controller. The controller will be created when the view of a FormViewController is loaded. This class is in charge of the view that represents the FormElement that created the controller.
