AUISwitch

public class AUISwitch: UIControl

The AUISwitch class declares a property and a method to control its on/off state. When the user manipulates the switch control (“flips” it) a valueChanged event is generated, which results in the control (if properly configured) sending an action message.

On
  • isOn

    A Boolean value that determines the off/on state of the switch.

    This property allows you to retrieve and set (without animation) a value determining whether the AUISwitch object is on or off.

    Declaration

    Swift

    public var isOn: Bool
  • setOn(_:animated:)

    Set the state of the switch to On or Off, optionally animating the transition.

    Declaration

    Swift

    public func setOn(_ on: Bool, animated: Bool)

    Parameters

    on

    true if the switch should be turned to the On position; false if it should be turned to the Off position. If the switch is already in the designated position, nothing happens.
    animated

    true to animate the “flipping” of the switch; otherwise false.
