  • init(clientId:service:environment:countryOfSite:legalEntity:clientIdSecret:keychainLocalAccessGroupName:accessTokenManager:sourceId:)

    Initializes a new AMCConnectCore instance with the provided specifications.

    Declaration

    Swift

    public init(clientId: String, service: AMCService,
            environment: AMCEnvironment, countryOfSite: String? = nil,
            legalEntity: String? = nil, clientIdSecret: String? = nil,
            keychainLocalAccessGroupName: String? = nil,
            accessTokenManager: AMCAccessTokenManager = .jwtToken,
            sourceId: String)

    Parameters

    clientId

    Client Identifier that identifies the application in backend
    service

    Service in which the framework will operate
    environment

    Environment in which the framework will operate
    countryOfSite

    The ISO 3166 - A2 code of country of site for Mobile App
    legalEntity

    Legal Entity for Mobile App
    clientIdSecret

    Secret to be used with the client ID in case your app requieres it. It is only necessary for Cloud and not for SCV.
    keychainLocalAccessGroupName

    Identifier of local keychain access group. Specify this parameter if application should support cross application login. Has to be such as main bundle identifier of the application same as first a ccess group specified in the project Capabilities
    accessTokenManager

    Indicates the type of token to be requested: default, ref or jwt
    sourceId

    The source or exact touch point from where the request is made to SCV by the recipient or silo

    Return Value

    A new AMCConnectCore instance.
