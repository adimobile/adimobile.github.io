  • init(clientId:service:environment:countryOfSite:legalEntity:clientIdSecret:keychainLocalAccessGroupName:accessTokenManager:)

    Initializes a new AMCConnectCore instance with the provided specifications.

  • Parameters:
  • clientId: Client Identifier that identifies the application in backend
  • service: Service in which the framework will operate
  • environment: Environment in which the framework will operate
  • countryOfSite: The ISO 3166 - A2 code of country of site for Mobile App
  • legalEntity: Legal Entity for Mobile App
  • clientIdSecret: Secret to be used with the client ID in case your app requieres it
  • keychainLocalAccessGroup: Identifier of local keychain access group. Specify this parameter if application should support cross application login. Has to be such as main bundle identifier of the application same as first a ccess group specified in the project Capabilities

  • accessTokenManager: Indicates the type of token to be requested: default, ref or jwt

    • Returns

    A new AMCConnectCore instance.
