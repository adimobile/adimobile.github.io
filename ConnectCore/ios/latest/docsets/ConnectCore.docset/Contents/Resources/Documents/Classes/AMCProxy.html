AMCProxy

public class AMCProxy: NSObject

Note that this method will work only for scv. Call this method if your app needs to set up the load proxy. This load proxy will be used to avoid the server down with a high number of requests. The proxy will return an http erroc code once the proxy gets the maximun of calls allowed. @param appId String containing the app id used for proxy authentication @param appKey String containing the key used for proxy authentication @param scvProxyUrl Url used for the scv proxy. @param pfProxyUrl Url used for the ping federate proxy. @return Builder to continue building connect client.
